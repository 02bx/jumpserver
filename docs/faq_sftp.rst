sftp 使用说明
-------------------------------------------------------
在Windows上使用 sftp 工具传输文件到 Linux 系统，默认的上传目录在 /tmp ，其他目录没有权限

.. image:: _static/img/faq_sftp_01.jpg

::

    # 连接成功后，可以看到当前拥有权限的资产，打开资产，然后选择系统用户，即可到资产的 /tmp 目录
    $ sftp -P2222 admin@192.168.244.144  # Linux 语法
    $ cd 你的资产
    $ cd 你的系统用户
    # 此处即是当前资产的 /tmp 目录
